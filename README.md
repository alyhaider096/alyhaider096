## 🚀 About Me

I build **AI-powered automation systems** that help businesses reduce manual work, handle customers faster, connect tools, and improve day-to-day operations.

My work focuses on combining **Agentic AI, RAG, vector databases, AI chatbots, workflow automation, backend APIs, CRM systems, and SaaS platforms** into complete systems — not just simple bots or one-step automations.
